Project Overview

For this assignment, I built a small healthcare monitoring system using an ESP32 microcontroller and FreeRTOS. The system tracks a patient’s heart rate and blood oxygen levels in real time, triggering alarms if the readings become dangerously high or low.

The project simulates a real-world critical care scenario where timing is very important. Heart rate and oxygen monitoring tasks have strict deadlines, so if the system reacts too slowly, it could put a patient at risk. There is also a button to acknowledge alarms, LEDs to indicate system status, and logging to keep track of vital readings over time.

This project helped me practice building a real-time system with multiple tasks, synchronization, and safety critical timing, giving hands-on experience similar to what engineers face in medical device development.
