Project Overview

For this assignment, I stepped into a fictional scenario where I was hired as a new software engineer at a tech company in Central Florida. My responsibility was to help create a real-time system — a type of system where reacting quickly is just as important as producing the right result.

To make the experience more realistic, the project focused on the healthcare technology field. Devices like heart monitors, insulin pumps, and pacemakers all depend on software that must react within strict time limits. If a patient’s heart rate spikes or drops suddenly, the system needs to respond almost immediately. These kinds of devices rely on real-time systems to help maintain safety, accuracy, and reliability.

Application 6 let me build a small real-time application of my own, giving me hands-on experience with tasks, timing, and predictable behavior — all key parts of embedded systems engineering.
